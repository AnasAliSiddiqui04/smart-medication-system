# Smart Medication System

Smart Medication System is a university project developed for the Theory of Automata course.

## Project Description
This project is a simple smart medication management system. It helps manage patients and their medications. The system uses automata-based pattern validation to check medication schedules.

## Features
- Add and manage patients
- Add medications with schedule patterns
- Automata-based pattern validation
- Streamlit-based frontend
- FastAPI-based backend

## Technologies Used
- Python
- Streamlit
- FastAPI

## How to Run the Project

### Backend
```bash
uvicorn main:app --reload
